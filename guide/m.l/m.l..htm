メーリングリスト開設のお知らせと参加のおすすめ

　

2003年9月17日　　

日本国際地図学会　　

　

　日本国際地図学会では「日本国際地図学会メーリングリスト（略称：jcaml）」を開設し、2003年9月17日から運用を始めております。

　このメーリングリストへの加入は、本会々員に限らず、だれでもできます。

　参加費（加入料・会費等）は不要です。接続料・通信料は利用者の負担となります。jcaml　は　Yahoo!　Groups　を利用しております（2004年2月9日から、Yahoo! eGroups が Yahoo!　Groups となりました.利用方法の変更についてはこちらのページをごらんください）。

　加入には申込みが必要です。(本会会員であっても申込が必要です。本会特別会員には特例がありますので、別途ご連絡を差し上げます。)

　申し込み方法は、『ウェブサイトからの参加』 と 『メールによるグループへの参加』の二種類があります。参加手続き後、メーリングリスト管理人から加入申込書を送付（eメール）し、所定事項記入後ご返送下さい。意志確認メールを［返信］後、jcaml　への参加ができます。詳しくは、以下の申し込み方法をごらんいただくか、メーリングリスト管理人(グループオーナー) jcaml-owner@yahoogroups.jp までお問い合わせ下さい。

　　

　　

“jcaml”参加方法

　“jcaml”への参加方法は、大きく分けて【１．ウェブサイトからの参加方法】と、【２．メールによるグループへの参加方法】の２つがあります。

１．ウェブサイトからの参加方法
　 　この場合、Yahoo! JAPAN ID の取得が必要です。以下の手順、及びYahoo! Groupsのページを参考にしてください。jcamlウェブページの機能 <過去に投稿されたメッセージの閲覧、メール配信方法の設定、ブリーフケースからのダウンロード、メールアドレスの変更など>が利用できる正攻法です。
２． メールによるグループへの参加方法
　 　この場合、Yahoo! JAPAN ID の取得、ニツクネームの登録、バスワードの設定などは不要です。メーリングリストの投稿と閲覧は、ウェブサイトからの参加方法と同様に利用できます。ただし、jcamlのウェブページの機能を利用することはできません。加入後に Yahoo! ID を取得することは可能です。とりあえず"jcaml"に加入したいという方には便利な簡易法です。

２．

 メールによるグループへの参加方法

　この場合、Yahoo! JAPAN ID の取得、ニツクネームの登録、バスワードの設定などは不要です。メーリングリストの投稿と閲覧は、ウェブサイトからの参加方法と同様に利用できます。ただし、jcamlのウェブページの機能を利用することはできません。加入後に Yahoo! ID を取得することは可能です。とりあえず“jcaml”に加入したいという方には便利な簡易法です。

　　　

 メール(Subscribe方式)の発信
　参加申込のアドレスに、“jcaml”に登録するメールアドレスからメールを送信してください(このメールに件名や本文を記入する必要はありません。メールは自動で処理されます)。
　　　　申込アドレス：　jcaml-subscribe@yahoogroups.jp
　　　　
確認メールの返信
　メールが正しく送信されると、“jcaml”に参加する意志の確認メールが届きます。
届いたメールにメールソフトの [返信] 機能を使って返信すると、参加申込が完了します。
“jcaml”は参加に承認が必要なため、ここで参加の承認待ちの状態となります。「加入申込書」の送付については、Ⅱ．をご覧ください。
　　
Ⅱ．加入申込書の送付と加入の承認

１．

 加入申込書の送付
　折り返し、「加入申込書」および「利用規約」が接続したメールアドレスに届きます。加入申込書に所定事項を記入し、メーリングリスト管理人(グループ管理者)まで返信します。 
　　返信アドレス：　jcaml-owner@yahoogroups.jp 

２．

 加入の承認
　申込書を審査し管理者が参加を承認すると、登録のメールアドレスに承認のメールが届きます。これで参加手続きが完了し、ただちに jcamlのご利用ができます。
　申込書の受領から承認までには２週間程度かかります。なお、加入申込書が相当期間を経過しても返送されない場合、参加の申込を取り消します。
　　

jcamlの運用基準については、メーリングリスト運用規程（ ＨＴＭＬ版 ・ PDF版 ）およびメーリングリスト利用規約（ ＨＴＭＬ版 ・ PDF版 ）をご覧ください。また、Yahoo! グループ　への改組に伴い、2003年２月９日以前にメーリングリストへ登録された方々の変更手続きについては、こちらのページをごらんください。

ご不明の点は、メーリングリスト管理人 jcaml-owner@yahoogroups.jp までお問い合わせください。
